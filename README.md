# PokeAPI PHP Wrapper

PHP Wrapper for [PokeAPI](http://pokeapi.co)

## Features

* Request Class
* Pokemon Class
* Type Class
* Move Class
* Ability Class
* Egg Group Class
* Description Class
* Sprite Class
* Game Class
